ア関係 “フライト” は, 更新時異状が発生することはない。
イ関係 “フライト” は, 自明でない関数従属が存在する。
ウ関係 “フライト” は, 情報無損失分解が可能である。
エ関係 “フライト” は, ボイス・コッド正規形の条件は満たしていない。

従業員番号と氏名と使用できるプログラム言語を管理するために, “従業員” 表及び “プログラム言語” 表を設計する。“プログラム言語を2種類以上使用できる従業員がいる。プログラム言語を全く使用できない従業員もいる。” という状況を管理する “プログラム言語” 表の設計として, 適切なものはどれか。ここで, 実線の下線は主キーを表す。

[従業員表]

アプログラム言語 (氏名, プログラム言語)
イプログラム言語 (従業員番号, プログラム言語)
ウプログラム言語 (従業員番号, プログラム言語)

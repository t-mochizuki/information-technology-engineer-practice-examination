ア何らかの方法で事前に利用者 ID と平文のパスワードのリストを入手しておき, 複数のシステム間で使い回されている利用者 ID とパスワードの組みを狙って, ログインを試行する。
イパスワードに成り得る文字列の全てを用いて, 総当たりでログインを試行する。
ウ平文のパスワードとハッシュ値をチェーンによって管理するテーブルを準備しておき, それを用いて, 不正に入手したハッシュ値からパスワードを解読する。
